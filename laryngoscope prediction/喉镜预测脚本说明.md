# 喉镜预测脚本说明

## 1. 脚本位置

脚本位于 [predict_larynscope.py](/c:/Users/徐国城/Desktop/喉镜图预测/predict_larynscope.py)。

它现在支持两种使用方式：

1. 直接运行脚本，弹出窗口自己选图片
2. 用命令行参数指定单张图片或整个目录

## 2. 脚本做了什么

脚本会完成下面几件事：

1. 加载 [store/repalexnet_heatmap_n100.pth](/c:/Users/徐国城/Desktop/喉镜图预测/store/repalexnet_heatmap_n100.pth) 权重
2. 调用 [model_repAlexnet.py](/c:/Users/徐国城/Desktop/喉镜图预测/larynscope/model_repAlexnet.py) 中的 `RepAlexNet`
3. 对输入图片进行分类预测
4. 用 [utils_heatmap.py](/c:/Users/徐国城/Desktop/喉镜图预测/larynscope/utils_heatmap.py) 中的 `GradCAM` 生成热力图
5. 为本次运行单独创建一个新输出目录
6. 将热力图、JSON 结果和文字汇总保存到这个新目录中

## 3. 默认运行方式

直接执行：

```powershell
python predict_larynscope.py
```

脚本会弹出文件选择框，你可以自己选一张或多张喉镜图片。

如果你想选整个文件夹，可以这样运行：

```powershell
python predict_larynscope.py --select-dir
```

## 4. 命令行方式

处理单张图片：

```powershell
python predict_larynscope.py --input store/larynscope/202012010042005.jpg
```

处理一个目录：

```powershell
python predict_larynscope.py --input store/larynscope
```

递归处理目录中的子目录图片：

```powershell
python predict_larynscope.py --input some_folder --recursive
```

指定本次运行的输出目录：

```powershell
python predict_larynscope.py --input store/larynscope --output-dir output/my_run
```

强制用 CPU：

```powershell
python predict_larynscope.py --device cpu
```

## 5. 输出目录规则

如果你没有手动指定 `--output-dir`，脚本会自动创建一个新的运行目录，格式类似：

```text
outputs/larynscope_run_20260412_170500/
```

这个目录里会包含：

- `heatmaps/`：本次生成的所有热力图
- `predictions.json`：结构化结果
- `predictions_summary.txt`：更容易直接查看的文字汇总

这样新热力图就不会再写到旧的 [store/store/heatmap](/c:/Users/徐国城/Desktop/喉镜图预测/store/store/heatmap) 里。

## 6. 输出结果内容

### 热力图

每张图片会生成一张对应的热力图，保存在本次运行目录下的 `heatmaps/` 中。

### JSON 结果

`predictions.json` 里每条记录包含：

- `image`：输入图片路径
- `predicted_class_id`：模型预测类别编号
- `predicted_label`：病变类型文字
- `confidence`：预测置信度
- `probabilities`：6 个类别的 softmax 概率
- `heatmap_path`：对应热力图路径

### 文字汇总

`predictions_summary.txt` 会按更直观的格式列出：

- 图片名
- 预测结果
- 置信度
- 热力图路径

## 7. 乱码处理

脚本启动时会主动把标准输出切到 UTF-8，并在 Windows 下尝试设置控制台编码，专门用来缓解中文输出乱码问题。

如果终端里仍然显示异常，优先查看：

- `predictions.json`
- `predictions_summary.txt`

这两个文件都是 UTF-8 保存，通常在编辑器里会正常显示。

## 8. 关于类别名称的说明

脚本里的中文标签映射，沿用了 [demo.py](/c:/Users/徐国城/Desktop/喉镜图预测/demo.py) 当前界面的显示逻辑：

- `0 -> 健康`
- `1 -> 喉鳞状细胞癌`
- `3 -> 炎症角化`
- 其余类别当前统一显示为 `不典型增生`

因为仓库里明确写出的中文映射只有这几项，所以脚本同时保留了 `predicted_class_id`。
后续如果你补齐 6 类的准确名称，只需要修改 [predict_larynscope.py](/c:/Users/徐国城/Desktop/喉镜图预测/predict_larynscope.py#L22) 里的 `DISPLAY_LABELS`。
