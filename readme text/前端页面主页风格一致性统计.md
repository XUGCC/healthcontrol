## 3. 主页风格判断口径

判断一个页面是否属于“主页风格”，主要参考以下特征：

- 是否使用 `hc-mobile-shell`
- 是否使用 `hc-screen`
- 是否使用 `hc-topbar`、`footer-bar` 或与主页一致的头部表达
- 是否复用 `hc-card-dark / hc-card-soft / hc-card-lime`
- 是否复用 `hc-pill-button-*`
- 是否延续主页的大圆角、浅绿渐变、模块卡片、摘要优先的页面结构
- 是否已经移除旧式 `uni-nav-bar + 白底列表/表单壳` 的页面骨架

返回能力的复查口径如下：

- 普通内页需要具备明确的左上返回入口
- 根入口页允许在“没有上级路由栈”时按条件隐藏返回入口
- 已主页化但此前未显式写 `showBack` 的页面，统一由 `hc-topbar` 默认返回能力兜底
