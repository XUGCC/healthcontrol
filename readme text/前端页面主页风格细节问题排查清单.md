# 前端页面主页风格细节问题排查清单

生成时间：2026-03-29  
最后更新：2026-03-29

## 1. 本轮结论

- `HealthControl.uniapp/pages/Front` 中，与主页风格细节相关的表单控件、选择面板、确认弹层、操作菜单、日期时间弹层，已完成两轮统一治理。
- 本轮重点处理了“弹层被卡片/固定按钮遮挡”“弹层像从卡片底部弹出”“时间滚轮对不准、显示值不正确”三类问题。
- 当前前端用户页中的弹层统一口径为：`底部弹出 + 高层级覆盖 + 安全区适配 + 超高内容可滚动`。
- 当前前端用户页中，不再保留 `type="center"` 的 `uni-popup`；与用户直接交互的自定义弹层已统一为从底部弹出。

## 2. 已完成项

### 2.1 旧版白色原生控件替换

- 已清理的系统原生交互包括：
  - `uni.showModal`
  - `uni.showActionSheet`
  - 页面内直接使用的 `picker / uni-datetime-picker / switch / uni-data-checkbox / uni-number-box`
- 上述交互此前已统一替换为主页风格组件或主页风格弹层。

### 2.2 弹层统一改为底部弹出

- 以下共用组件已统一为底部弹层：
  - `hc-dialog.vue`
  - `hc-select-sheet.vue`
  - `hc-action-sheet.vue`
  - `hc-time-field.vue`
  - `hc-date-field.vue`
- 以下仍保留页面内自写 `uni-popup` 的场景，也已统一到底部弹层样式并补齐同样的层级/滚动策略：
  - `DietFoodDetail.vue`
  - `ScienceDetail.vue`
- 本轮已补齐：
  - 顶部拖拽条视觉锚点
  - `max-height` 限制
  - `overflow-y: auto`
  - 底部安全区留白

### 2.3 弹层遮挡与裁切问题修复

- 已在 `App.vue` 中统一提升 `uni-popup` 全局层级，避免被以下元素压住：
  - 页面底部固定 CTA
  - 固定输入栏
  - `footer-bar`
  - 页面内高层级悬浮区
- 这类问题对应的典型页面包括：
  - `HealthReminder.vue`
  - `DietFoodDetail.vue`
  - `ScienceDetail.vue`
  - 以及所有复用 `hc-dialog / hc-select-sheet / hc-time-field / hc-date-field / hc-action-sheet` 的页面
- 原先“弹层展开后像被框在卡片里”“底部按钮压在弹层上面”“部分日期和选项点不到”的问题，已按统一层级方案处理。
- 追加修复：`App.vue` 的 `hc-reveal-up` 动画已彻底移除 `transform` 位移动画（不仅结束态，动画过程也不使用 `transform`），避免 `position: fixed` 弹层在动画阶段被上层卡片容器当作定位上下文，确保真正从整屏底部弹出，而不是从卡片底部弹出。
- 追加修复：`App.vue` 中主页卡片容器（`hc-card-soft / hc-card-lime / hc-card-dark`）已改为 `overflow: visible`，不再裁切内部弹层；弹层全局层级同步提升为 `99999`，确保弹窗始终处于最上层，不与其他卡片重叠。
- 追加修复：`App.vue` 已移除 `.hc-screen` 的 `z-index` stacking context，并将背景装饰层调整为 `z-index: -1`（配合 `hc-mobile-shell` 的 `isolation: isolate`），解决“页面固定按钮在根层级压住弹层”的场景，确保弹层能压过页面内固定 CTA。

### 2.4 时间弹窗与日期时间弹窗修复

- `hc-time-field.vue` 已修复：
  - 打开时强制按当前值重新同步滚轮位置
  - 通过 `picker key` 重建滚轮，避免出现显示停留在旧位置
  - 选中指示区补齐真实样式，解决“高亮线没对准数字”的问题
  - 底部预览值改为严格跟随当前滚轮索引，解决“下面显示的时间和上面不一致”的问题
  - 分钟步进改为按真实可选值映射，避免索引和分钟文本错位
- `hc-date-field.vue` 已修复：
  - 日期时间场景下，时间滚轮同样会在打开时按当前值重新同步
  - 选中指示区样式已补齐
  - 时间滚轮与摘要预览值保持一致
  - 日期面板与时间面板统一纳入底部弹层，不再受卡片容器裁切
