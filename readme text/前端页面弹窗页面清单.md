- 已接入统一弹层底座的共享组件：
  - `hc-dialog`
  - `hc-action-sheet`
  - `hc-select-sheet`
  - `hc-date-field`
  - `hc-time-field`
- 页面级单独重写：
  - `HealthControl.uniapp/pages/Front/DietFoodDetail.vue`
  - `HealthControl.uniapp/pages/Front/ScienceDetail.vue`
- 改造目标：
  - 弹层提升为页面最高层，避免被卡片、顶部文字、悬浮操作区遮挡
  - 当前实现已调整为优先保障层级正确的全屏顶层弹层，先确保不会再被页面内容覆盖
