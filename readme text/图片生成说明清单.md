# 图片生成说明清单

- 生成依据：`food-db-snapshot.md`、`image-tables-snapshot.md`
- 适用范围：`tlaryngealfood`、`totolaryngologyhospitaldoctor`、`thealthscience`、`tlaryngoscopephoto`
- 用途：给外部生图平台或设计师提供统一的图片名与生成描述

## 通用要求

- 所有图片默认要求：无文字、无水印、无品牌 logo、无 UI 截图元素、无边框
- 整体风格优先：写实感强、自然光、干净构图、主体明确、颜色不过度夸张
- 若用于小程序卡片：主体尽量位于画面中部偏左或中部，避免重要内容贴边
- 文件命名建议优先沿用下表，后续替换数据库时更容易对应
